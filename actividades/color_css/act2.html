Background Color - Color de fondo

Si no se especifica un color de fondo, el fondo es transparente

CSS trata cada elemento HTML como un cuadro. Se puede especificar un color de fondo de las mismas tres formas en que puede especificar colores de primer plano: Valores RGB, Códgio Hexadecimal y Nombre de Colores. También hemos utilizado la propiedad padding para separar el texto de los bordes de los cuadros. Esto hace que sea más fácil de leer.
