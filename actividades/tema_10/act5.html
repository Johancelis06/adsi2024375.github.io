YouTube video ID YouTube mostrará un ID (como EF81Pb5IWoU), al guardar (o reproducir) un vídeo. Puede utilizar este ID y hacer refrencia a su vídeo en el código HTML.

Reproducción de un vídeo de YouTube en HTML:
Para reproducir el vídeo en una página web, haga lo siguiente:

  • Sube el vídeo a YouTube
  • Anote el ID de vídeo
  • Defina un elemento <ifram> en su página web
  • Deje que el atributo src apunte la URL del vídeo
  • utilice los atributos width y height para especificar la dimensión del reproductor
