En HTML5 no es necesario proporcionar valores para todos los atributos, como los controles, reproducción automática y atributos de bucle utilizados con el elemento <video>. Estos atributos son como interruptores de encendido/apagado. Si el atributo está presente, activa esa opción, Si se omite el atributo, la opción está desactivada. Si el navegador no amite el elemento <video> de apertura y las etiquetas de </video> de cierre

preload este atributo le indica al navegador que hacer cuando se carga la página. Puede tener uno o tres valores:
none: el navegador no debe cargar el vídeo hasta que el usuario presiona play.
auto: el navegador debe descragar el video cuando se carga la página
metadta: el navegador debe simplemente recopilar información como el tamño, el primer fotograma, la lista de pistas y la duración.
