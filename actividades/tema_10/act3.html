<audio> HTML5 introdujo el elemento <audio> para incluir archivos de audio en sus páginas. Al igual que con el vídeo HTML5, los navegadores esperan diferentes formatos para el audio. El elemento <audio> tiene un número de atributos que le permiten controlar la reproducción de auido:

src: este atributo especifica la ruta de acceso al archivo de audio.

controls: este atributo indica si el reproductos debe mostrar los controles si no utiliza este atributo, no se mostrarán controles de forma predeterminada. También puede especificar sus propios controles mediante Javascript.

autoplay: La presencia de este atributo indica que el audio debe empezar a reproducirse automáticamente. (SE considera una mejor práctica permitir que los visitantes alijan reproducir audio.)

loop: este atributo especifica que la pista de audio debe reproducirse de nuevo una vez que haya finalizado
