Cada elemento HTML puede llevar el atributo id. Se utiliza para identificar de forma única ese elemento de otros elementos de la página. Su valor debe comenzar con una letra o un guión bajo (no un número o cualquier otro carácter). Es importante que no haya dos elementos en la misma página qque tienen el mismo valor para sus atributos ID (de lo contrario, el valor ya no es único)

dar a un elemento una identidad única le permite darle un estilo diferente que cualquier otra instancia del mismo elementode la página

Por ejemplo, es posbile que desee asignar un párrafo dentro de la página (qui´zas un párrafo que contiene una cotización de extracción) un estilo diferente que todos los demás párrafos. En el ejemplo, el parrafo con el atributo ID cuyo valor es pullquote se hace en mayúculas usando CSS
